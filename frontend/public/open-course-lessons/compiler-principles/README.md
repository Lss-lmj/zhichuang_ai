# 编译原理 · 交互式教学演示

一套面向大学"编译原理"课程课堂投屏使用的可交互 HTML 演示合集，共 12 个课节 + 1 个导航首页。

## 快速开始

在浏览器中直接双击打开 `index.html` 即可。全部页面无任何 CDN、无网络请求、纯本地离线运行。

## 通用键盘约定（全部页面统一）

| 按键 | 功能 |
|---|---|
| `←` | 上一步 |
| `→` | 下一步 |
| `Space` | 播放 / 暂停自动播放 |
| `R` | 重置到第 1 步 |

底部控制条同时提供按钮：**上一步 / 下一步 / 自动播放 / 步骤圆点 / 重置**；右下角悬浮"速度"按钮循环切换 `0.5× / 1× / 2×`。

## 教师二次改造建议

- **主题演示**：每份子页 JS 顶部都集中定义了数据结构（如 `NFA_NODES / DFA_EDGES / STEPS / LL_TRACE / TABS` 等）并配中文注释；改换示例（正则、文法、代码片段）时只需替换这些数据数组，视图与控制代码可直接复用。
- **配色/字体/组件类名**：统一定义在 `DESIGN_SPEC.md`，全部通过 CSS 变量在页面头部集中管理，改一处即可全站生效。
- **新增课节**：新增页面应围绕该课节的知识对象单独设计交互，不把现有页面简单改标题后复用。新增页面需要保持目录返回链接，并在 `index.html` 的卡片网格与 Pipeline Bar 中新增对应入口。
