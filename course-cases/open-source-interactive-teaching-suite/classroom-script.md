# 开源课堂演示脚本

## 演示目标

展示智创Agent 不只是项目管理系统，也可以作为计算机专业课程的交互式授课平台。教师可以直接在课堂上选择课程、章节和课节，使用动画讲解抽象机制，布置实验任务，并通过项目或实验产出进行讲评。

## 7 分钟演示结构

### 0:00-0:40 进入开源课堂

1. 登录系统。
2. 从左侧导航进入“开源课堂”。
3. 说明当前页面覆盖开源创新大赛八门课程、32 章、96 个课节。

讲解重点：

- 八门课程不是八套孤立系统，而是一套统一的交互教学框架。
- 课程资料、实验任务、项目提交和教师讲评可以沉淀为可开源案例包。

### 0:40-1:40 展示计算机系统导论

1. 选择“计算机系统导论”。
2. 选择“函数调用与栈帧”章节。
3. 选择“栈帧布局”或“缓冲区越界”课节。
4. 播放指令执行与栈帧变化动画。
5. 切换到“实操”，展示函数调用栈、边界写入风险和模拟器日志任务。

讲解重点：

- 抽象的寄存器、栈帧和内存变化可以在课堂上同步展示。
- 系统可以解释汇编片段、追踪寄存器状态、定位风险代码。

### 1:40-2:40 展示编译原理

1. 选择“编译原理”。
2. 选择“中间代码与优化”章节。
3. 选择“三地址码”课节。
4. 展示 source、token、AST、symbol、IR、optimize 的分步过程。
5. 切换“讲评”，说明如何评价解析器代码和 IR 输出。

讲解重点：

- 学生不只写代码，还要提交 AST 输出、符号表快照和 IR 结果。
- 教师可以根据共性错误讲解文法、符号作用域和中间表示。

### 2:40-3:40 展示数据库系统概论

1. 选择“数据库系统概论”。
2. 选择“查询优化”章节。
3. 选择“EXPLAIN 读法”或“索引选择”课节。
4. 展示 SQL 到执行计划的动画。
5. 说明项目管理场景建模、查询脚本和索引优化实验。

讲解重点：

- 数据库课程可以和真实项目开发结合。
- 系统可以分析 SQL、解释执行计划、给出索引建议。

### 3:40-4:40 展示操作系统与 AI 基础设施

1. 选择“操作系统内核构建”，进入“内存管理”章节，展示页表映射。
2. 选择“AI 基础设施”，进入“分布式通信”章节，展示 AllReduce 通信同步。

讲解重点：

- 这两个方向适合用动画降低理解门槛。
- 课程实验可以从“看懂路径”逐步过渡到“补全代码”。

### 4:40-5:40 展示软件工程方向闭环

1. 选择“软件工程”。
2. 选择“项目管理与讲评”章节。
3. 选择“项目分析报告”课节。
4. 展示需求、设计、代码、测试、交付、复盘闭环。
5. 回到项目管理模块，说明学生可以上传项目并生成分析报告。
6. 说明教师看板如何汇总班级共性问题。

讲解重点：

- 软件工程方向已经和系统主线打通。
- 学生项目、团队分工、报告、改进事项和教师讲评构成完整教学闭环。

### 5:40-6:40 展示课堂产出与评价

1. 展示每门课程的章节目录、课节列表、实验任务、课堂产出和评价口径。
2. 说明这些内容可以作为开源教学案例包发布。

讲解重点：

- 不是只做动画，而是面向课程实验和评价。
- 每门课程都能形成可提交、可讲评、可复用的材料。

### 6:40-7:00 总结

一句话总结：

> 智创Agent 面向开源创新大赛不是单点功能，而是一套覆盖计算机专业八门课程、32 章、96 个课节的交互式开源教学案例平台；教师能直接用于课堂，学生能完成实验并提交项目证据。

补充说明：96 个课节均可进入独立交互页面，每节页面包含关键问题、过程演示、实操任务、课堂互动、作品证据和评价标准。

## 课堂使用建议

| 课堂环节 | 使用方式 |
| --- | --- |
| 课前导入 | 教师选择课程、章节和课节，进入对应课节的独立交互页面 |
| 课中实操 | 学生按实验任务完成代码、日志、截图或报告 |
| 课中提问 | 使用知识库问答解释概念和错误原因 |
| 课后提交 | 学生在项目管理中上传代码或实验材料 |
| 课堂讲评 | 教师查看项目报告、共性问题和评价口径 |

## 后续扩展

1. 为每门课程补充独立实验任务书。
2. 为数据库、系统导论、编译原理方向增加可运行样例代码。
3. 将 HTML 动画拆分为可嵌入课件的独立组件。
4. 把学生实验提交接入统一项目分析报告。
