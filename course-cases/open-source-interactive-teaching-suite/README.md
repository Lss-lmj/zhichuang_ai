# 开源创新大赛八门课程交互式教学案例套件

本案例套件面向开源创新大赛八个方向，目标不是只提交课程文档，而是让教师可以直接在课堂上使用智创Agent 进行概念讲解、实验指导、项目提交和课堂讲评。

## 套件定位

智创Agent 在开源创新大赛中的表达应升级为：

> 面向计算机专业核心课程的交互式开源教学案例与智能体辅助授课平台。

首版重点覆盖八门课程：

1. 深度学习框架与编程。
2. 计算机系统导论。
3. 编译原理。
4. 数据库系统概论。
5. 操作系统内核构建。
6. 计算思维相关方向。
7. 软件工程。
8. AI 基础设施。

每门课程包含 4 章，每章 3 节，首版共 32 章、96 个课节。每个课节均包含教学重点、过程可视化、课堂互动、实验任务、提交证据和系统支持能力。

前端开源课堂采用分层页面结构。教师先进入课程目录，再进入课程详情、章节详情，最后打开具体课节。任意课节页面都会呈现该节自己的课节编号、关键问题、过程演示、实操任务、课堂互动、作品证据和评价标准。

- 课程目录页：展示八门课程入口和课程规模。
- 课程详情页：展示课程章节和课程材料栈。
- 章节详情页：展示本章课节、课节重点和课时信息。
- 课节交互页：展示过程动画、实操任务、作品证据和评价标准。

## 课堂使用方式

教师可以在系统前端进入“开源课堂”模块，按“课程 -> 章节 -> 小节”切换授课场景：

- 授课演示：展示抽象过程的分步动画，例如自动求导、栈帧变化、SQL 执行计划、进程调度。
- 实操指导：展示实验任务、提交材料和系统支持能力。
- 课堂讲评：展示评价口径、共性问题和后续练习建议。

## 交付材料

| 文件 | 用途 |
| --- | --- |
| `course-syllabus.md` | 八门课程、32 章、96 节的章节化教学大纲 |
| `direction-matrix.md` | 八方向课程案例、实验任务、智能体能力和交互动画设计矩阵 |
| `classroom-script.md` | 教师课堂演示脚本，可用于录制演示视频或现场答辩 |
| `index.html` | 可直接打开的静态 HTML 交互演示样例 |

## 与软件工程案例包的关系

`course-cases/software-engineering-project-practice/` 是软件工程方向的 P0 详案，已经和系统中的项目管理、项目分析、教师看板形成较强闭环。

本套件负责补齐其他七门课程，并把八个赛题方向统一为“可课堂使用”的交互式授课能力。后续可以按比赛侧重点，把某一门课程继续扩展为更完整的课程实验包。
