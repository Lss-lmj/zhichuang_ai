# 生成样例：项目实践材料

## 项目：RAG 文档问答助手

### 项目目标

实现一个可以基于课程 Markdown 文档回答问题的小型 RAG 系统。

### 功能要求

1. 上传或读取 Markdown 文档。
2. 将文档按标题或长度切片。
3. 根据用户问题检索相关片段。
4. 基于片段生成回答。
5. 展示引用来源。
6. 资料不足时明确说明无法确认。

### 交付物

- 项目代码。
- README。
- 依赖文件。
- 至少 3 个测试问题。
- 输出示例。
- 项目复盘。

### 评价 Rubric

| 维度 | 权重 |
|---|---:|
| 文档解析和切片 | 20 |
| 检索效果 | 20 |
| 回答生成和引用 | 25 |
| 工程规范 | 20 |
| 测试和说明 | 15 |
