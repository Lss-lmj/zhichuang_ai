# 学习效果评估模板

## 评估输入

| 来源 | 示例 |
|---|---|
| 学习任务 | 完成 RAG 概念阅读 |
| 练习结果 | 向量检索题目正确率 70% |
| 项目报告 | README 缺失、测试不足 |
| 学生反馈 | 时间不足，希望任务更细 |

## 评估输出

| 项目 | 内容 |
|---|---|
| 掌握情况 | 已理解 RAG 基本流程，测试和部署仍偏弱 |
| 画像变化 | AI 应用能力置信度提升，工程规范短板保留 |
| 下一步资源 | 测试样例编写、README 模板、部署检查清单 |
| 路径调整 | 将原 2 周项目任务拆成 4 个小任务 |

## 指标

| 指标 | 说明 |
|---|---|
| 资源完成率 | 已完成资源 / 推荐资源 |
| 练习正确率 | 正确题数 / 总题数 |
| 项目改进完成率 | 已完成改进事项 / 改进事项总数 |
| 画像证据覆盖率 | 有证据的画像维度 / 全部画像维度 |
