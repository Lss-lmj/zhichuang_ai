# 多智能体资源生成工作流

## Agent 分工

| Agent | 角色 | 输出 |
|---|---|---|
| 画像 Agent | 抽取学生画像、短板和偏好 | 画像维度、置信度、证据 |
| 课程 Agent | 检索课程知识库和项目案例 | 知识点、引用来源、推荐材料 |
| 资源 Agent | 生成讲解文档和拓展阅读 | 文档、资源卡片 |
| 练习 Agent | 生成题目和检查点 | 选择题、简答题、代码题 |
| 项目 Agent | 生成实操案例和项目任务 | 代码练习、项目任务书 |
| 评估 Agent | 检查资源是否匹配画像 | 学习效果摘要、路径调整 |

## 工作流

```text
学生输入学习目标
  -> 画像 Agent 抽取画像
  -> 课程 Agent 检索知识库
  -> 资源 Agent 生成讲解文档
  -> 练习 Agent 生成练习题
  -> 项目 Agent 生成实操案例
  -> 评估 Agent 检查资源难度和覆盖
  -> 保存学习路径和资源卡片
```

## 状态字段

| 字段 | 说明 |
|---|---|
| `student_id` | 学生 ID |
| `goal` | 学习目标 |
| `profile_dimensions` | 画像维度 |
| `resource_types` | 需要生成的资源类型 |
| `citations` | 引用来源 |
| `generation_status` | 生成状态 |
| `review_notes` | 评估 Agent 反馈 |
