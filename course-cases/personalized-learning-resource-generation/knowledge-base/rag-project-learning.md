# RAG 文档问答项目学习资料

## 学习目标

完成一个最小可运行的 RAG 文档问答项目，理解文档解析、切片、检索、生成和引用展示的基本流程。

## 核心知识点

1. 文档解析。
2. 文本切片。
3. Embedding。
4. 向量检索。
5. 提示构造。
6. 引用来源展示。
7. 答案边界说明。

## 推荐项目任务

学生可以先实现一个 Markdown 文档问答器：

1. 读取本地 Markdown 文件。
2. 按标题切片。
3. 用简单关键词检索或向量检索找到相关片段。
4. 调用模型生成回答。
5. 返回引用片段标题和路径。

## 常见问题

| 问题 | 建议 |
|---|---|
| 回答编造资料 | 增加引用校验和资料不足说明 |
| 检索结果不准 | 加入关键词召回和重排序 |
| 文档太长 | 优先按章节切片，再按长度兜底 |
