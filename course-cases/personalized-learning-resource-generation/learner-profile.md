# 对话式学习画像设计

## 示例输入

> 我是计算机大二学生，学过 Python 和数据库，算法一般。最近想做一个 RAG 文档问答项目，但不知道应该先学什么，也不太会写测试。

## 画像抽取结果

| 维度 | 结果 | 证据 |
|---|---|---|
| 知识基础 | Python、数据库入门 | 学生自述 |
| 算法能力 | 基础偏弱 | 学生自述“算法一般” |
| 工程实践 | 项目经验不足，测试能力不足 | 学生自述“不太会写测试” |
| AI 应用能力 | 想学习 RAG 项目 | 学生目标 |
| 学习目标 | 完成 RAG 文档问答项目 | 学生目标 |
| 资源偏好 | 项目实操优先 | 从“想做项目”推断 |
| 易错点 | 测试、项目拆解、学习顺序 | 学生困难描述 |
| 项目经历 | 尚未提供完整项目 | 证据不足 |

## 更新规则

1. 项目分析报告可提升工程实践维度置信度。
2. 练习题结果可更新知识基础和易错点。
3. 学生反馈“时间不足”会影响资源数量和任务粒度。
4. 学生完成项目后，系统将项目报告同步为画像证据。
