# 教师使用指南

## 一、课前准备

1. 在智创Agent 管理端导入课程、班级和学生基础数据。
2. 导入课程任务书、Rubric、项目案例和 FAQ。
3. 创建课程项目，设置提交说明和项目来源。
4. 向学生说明项目提交范围、AI 使用边界和评价口径。

## 二、项目分析流程

1. 学生提交项目后，系统自动生成项目分析报告。
2. 教师进入项目学情看板查看班级概览。
3. 教师按维度查看班级项目完成情况。
4. 教师打开单个项目报告，查看问题、亮点和证据片段。
5. 教师导出或整理课堂讲评材料。

## 三、课堂讲评建议

教师可围绕以下结构讲评：

1. 本次项目整体完成情况。
2. 班级共性问题。
3. 优秀项目做法。
4. 典型工程问题。
5. 下一阶段改进任务。

## 四、看板关注指标

| 指标 | 说明 |
|---|---|
| 提交数量 | 判断班级项目提交覆盖情况 |
| 维度均分 | 观察需求、设计、代码、测试、文档等能力分布 |
| 共性问题 | 支撑课堂讲评和补充实验安排 |
| 项目报告列表 | 快速进入单个学生或团队项目 |
| 改进事项 | 观察学生后续迭代方向 |

## 五、教学改进记录

建议每次课程项目讲评后记录：

- 班级名称。
- 项目主题。
- 参与学生数量。
- 项目提交数量。
- 主要共性问题。
- 本次讲评重点。
- 下次课程补充内容。
