# 软件工程项目实践课程案例包

本案例包服务智创Agent 的开源创新大赛方向，面向“软件工程项目实践”课程，提供可复用的课程项目任务、评价 Rubric、教师讲评流程、学生提交流程和知识库样例。

## 适用场景

- 软件工程课程项目。
- 数据库/Web 应用开发综合实践。
- 计算机专业项目制教学。
- 高仿真课堂项目讲评与形成性评价。

## 与智创Agent的关系

本案例包是课程内容层，智创Agent 是智能化支持系统：

1. 学生根据任务书完成项目并上传到项目中心。
2. 智创Agent 生成项目分析报告、证据片段和改进事项。
3. 教师通过项目学情看板查看班级共性问题和单个项目报告。
4. 课程资料、Rubric 和 FAQ 可进入知识库，支撑可追溯问答。

## 首版验收

| 验收项 | 标准 |
|---|---|
| 课程案例完整性 | 包含课程目标、任务书、Rubric、教师指南和学生指南 |
| 项目实践闭环 | 学生能按任务提交项目，系统能生成项目分析报告 |
| 教师教学支持 | 教师能查看班级项目学情、单个项目报告和讲评建议 |
| 可复用性 | 案例包不绑定学校特定数据，可迁移到同类课程 |
| 可验证性 | 至少形成 3 个项目报告样例和 1 份课堂运行记录 |
