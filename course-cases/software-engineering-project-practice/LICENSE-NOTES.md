# 开源协议与依赖说明

本文件用于说明软件工程项目实践课程案例包的开源复用边界。

## 案例包内容

本目录中的课程大纲、任务书、Rubric、教师指南、学生指南、知识库样例和运行记录模板，建议按仓库最终选择的开源协议发布。

## 第三方依赖

课程案例包本身不强绑定第三方代码依赖。智创Agent 系统实现涉及的前后端框架、数据库、RAG 组件和 Agent 编排工具，以仓库根目录、`backend/`、`frontend/` 中的依赖清单为准。

## 复用建议

其他课程复用本案例包时，应至少调整：

1. 课程名称和教学周期。
2. 项目主题和交付要求。
3. Rubric 权重。
4. 示例项目和知识库资料。
5. 课堂运行记录中的学校、班级和教师信息。

## 数据边界

案例包不包含学校特定账号、真实班级数据或个人信息。真实教学使用时，应由课程负责人自行配置课程、班级和学生数据访问范围。
