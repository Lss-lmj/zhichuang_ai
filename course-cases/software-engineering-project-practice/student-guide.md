# 学生使用指南

## 一、项目提交

1. 登录智创Agent。
2. 进入项目中心。
3. 创建课程项目。
4. 填写项目名称、来源、绑定课程、作者和团队分工。
5. 上传 zip 项目包、公开仓库链接或项目说明。
6. 提交后等待项目分析完成。

## 二、查看项目报告

项目报告包含：

- 功能完成度。
- 代码结构。
- 工程规范。
- 能力证据。
- 项目亮点。
- 主要问题。
- 改进事项。

报告中的评分是基于提交物证据的相对画像，用于帮助学生改进项目，不作为真实能力的绝对判断。

## 三、保存改进事项

学生可将报告中的建议保存为执行事项，例如：

- 补充 README 启动说明。
- 增加依赖文件。
- 完善接口错误处理。
- 补充基础测试样例。
- 调整模块边界。

## 四、重新提交

完成修改后，学生可以重新提交项目材料，形成新的项目报告。历史报告用于观察项目迭代过程。

## 五、AI 使用边界

学生可以使用智创Agent 辅助理解问题、检查项目和规划改进，但应独立完成课程项目的核心设计和实现。
