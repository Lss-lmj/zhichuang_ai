# 学生项目任务书

## 一、项目主题

学生可从以下方向选择项目，也可在教师同意后自拟题目：

1. 课程项目管理系统。
2. 校园服务预约系统。
3. 知识库问答系统。
4. 项目协作管理工具。
5. 数据可视化看板。
6. AI 学习助手。

## 二、项目交付要求

| 阶段 | 交付物 | 要求 |
|---|---|---|
| 需求阶段 | 需求说明 | 用户场景、功能清单、优先级、边界说明 |
| 设计阶段 | 设计说明 | 模块划分、数据模型、接口清单、页面流程 |
| 实现阶段 | 代码项目 | 可运行核心功能、依赖说明、入口说明 |
| 测试阶段 | 测试记录 | 基础测试样例、运行截图或测试输出 |
| 总结阶段 | 项目说明 | README、团队分工、问题与改进计划 |

## 三、提交方式

学生在智创Agent 项目中心提交：

1. 项目名称。
2. 绑定课程或实践场景。
3. 项目作者和团队分工。
4. zip 项目包、公开仓库链接或项目说明。
5. README、依赖文件、测试说明等辅助材料。

## 四、AI 支持边界

智创Agent 可用于：

- 检查需求和设计是否完整。
- 分析代码结构、工程规范和文档完整度。
- 生成项目分析报告和改进事项。
- 辅助教师讲评班级共性问题。

智创Agent 不用于：

- 替学生直接完成课程项目。
- 生成虚假项目经历、虚假测试结果或虚假竞赛材料。
- 代替教师作出唯一性终结评价。
