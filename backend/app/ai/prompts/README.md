# Prompt Assets

Versioned prompt templates for core workflows live here.

Recommended structure:

```text
prompts/
  assignment_analysis/
    v1.md
    schema.json
    eval_cases.jsonl
  learning_plan/
  competition_recommendation/
  team_recommendation/
```

Each prompt asset should define:

- Task goal.
- Input schema.
- Output schema.
- Citation rules.
- Failure rules.
- Evaluation cases.
