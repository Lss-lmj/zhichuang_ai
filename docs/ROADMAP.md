# 开发路线图

## M0：仓库与文档基线

- 初始化 Git 仓库。
- 建立前后端目录。
- 完成立项文档、架构设计、HTML PPT。
- 建立开发 SOP、API 草案、数据模型草案。

## M1：后端基础能力

- FastAPI 应用启动。
- 健康检查接口。
- 用户、课程、班级、项目基础模型。
- SQLite 本地开发数据库。
- Alembic migration。
- 基础权限中间件。

## M2：项目代码分析 MVP

- 项目上传或仓库链接提交。
- `CodeAnalysisGraph` 首版流程。
- 多维度评分和证据输出。
- 学生项目分析报告。
- 教师查看单个项目报告。

## M3：RAG 知识库 MVP

- 文档上传和解析。
- 资料切分、embedding、入库。
- 知识库检索接口。
- 引用来源返回。
- 首批课程、竞赛、项目案例资料入库。

## M4：教师学情诊断看板

- 班级提交统计。
- 维度分布图。
- 共性问题摘要。
- 项目报告列表。
- 项目讲评建议。

## M4.5：开源课堂与八门课程教学案例套件

- 开源课堂前端页面。
- 八门课程章节化教学大纲。
- 八方向交互式教学案例矩阵。
- 8 门课程、32 章、96 节课节内容。
- HTML 动画和过程可视化演示样例。
- 软件工程项目实践课程案例包。
- 课程任务书、教师指南、学生指南。
- 项目 Rubric 和项目报告样例。
- 高仿真课堂运行记录。
- 教师课堂讲评材料模板。

## M5：学生成长规划与推荐

- 能力画像汇总。
- 学习计划生成。
- 用户确认后修改计划。
- 竞赛推荐。
- 组队推荐。
- 阶段反馈。

## M5.5：个性化资源生成与多智能体学习

- 对话式学习画像样例。
- 多 Agent 资源生成工作流。
- 讲解文档、思维导图、练习题、拓展阅读、代码实操、项目实践材料。
- 资源卡片和学习路径推送。
- 学习效果评估记录。

## M6：公网访问环境与学校使用版本

- 学校账号。
- 示例课程和项目数据。
- 容器化部署。
- 前端构建部署。
- 操作手册和展示脚本。

## P1：鸿蒙端客户端

- ArkTS + ArkUI 客户端。
- 登录和个人首页。
- 学习路径查看。
- 任务提醒。
- 项目报告查看。
- 竞赛建议查看。
